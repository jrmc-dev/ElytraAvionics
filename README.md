# ElytraAvionics

A Minecraft Fabric mod that adds a HUD and altitude callout sounds for elytra flight.

## Features

- **HUD display**: shows your current vertical speed and height above ground while gliding
- **Altitude callouts**: audio cues when descending through 5, 10, 20, 30, 40, 50, 60, 70, and 100 blocks above ground
- **Master warning**: continuous audio alert when your elytra durability drops below 10

## Requirements

- Minecraft 1.21.11
- Fabric Loader ≥ 0.19.2
- Fabric API

## Building from source

```bash
./gradlew build
```

The built jar will be in `build/libs/`.
